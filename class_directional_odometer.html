Detailed Description

A class to represent directional Odometers.

These sensors typically expose three signals lines, as opposed to two in the directionless counterparts. Depending on the state of this third signal, the direction of movement can be inferred.

void DirectionalOdometer::update ( )
overridevirtual

Conducts the distance and speed measurements.

Updates the current dt with the time difference between the last two pulses and increases the pulse counter. Do not call it directly in your sketch! Instead, pass it inside a lambda as an argument to attach. Example: ``` const int ODOMETER_PIN = 2; Odometer odometer; odometer.attach(ODOMETER_PIN, [](){odometer.update();}); ```

virtual bool Odometer::attach ( uint8_t  pin,
void(*)()  callback 
)
pure virtual

Initializes the Odometer to receive pulses from the specified pin.

Parameters
pinThe pin to receive pulses from, which must support external hardware interrupts
callbackThe function to be run on each incoming pulse. Please use the following as an argument: [](){yourOdometerInstance.update();}
Returns
false if an error occurred, i.e. invalid pin

You must run this function before making any calls to your Odometer or you will not get any usable distance or speed from it.

Example:

const unsigned short LEFT_ODOMETER_PIN = 2;
const unsigned long PULSES_PER_METER = 80;
DirectionlessOdometer leftOdometer(PULSES_PER_METER);
void setup() {
leftOdometer.attach(LEFT_ODOMETER_PIN, []() {
leftOdometer.update();
});
}
